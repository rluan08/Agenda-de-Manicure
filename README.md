# Agenda de Manicure

Sistema simples para organizacao de agendamentos de clientes.

## Funcionalidades

- Cadastro de cliente
- Escolha de data e horario
- Tipo de servico
- Visualizacao em calendario
- Armazenamento local

## Tecnologias

- HTML
- CSS
- JavaScript
- PWA

## Como executar

1. Baixe o repositorio
2. Abra o arquivo index.html no navegador
